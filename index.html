How To Use

Get started with MOG-AUTOCLICKER in just a few simple steps

1

System Requirements

Before you begin, make sure your system meets these requirements:

  • Python 3.11 or higher (recommended)
  • Windows operating system
  • Basic knowledge of running Python applications
2

Install Required Libraries

Open Command Prompt (CMD) and run the following command to install all required dependencies:

pip install keyboard mouse pyautogui tk ttkthemes pywin32 threading time messagebox ttk tkinter
3

Download and Extract

Download the latest version of MOG-AUTOCLICKER from the download page and extract the ZIP file to a location of your choice.

4

Run the Application

Navigate to the extracted folder and run the main Python script:

  • Double-click the Python file to run it directly, or
  • Open Command Prompt, navigate to the folder, and type python mog_autoclicker.py
