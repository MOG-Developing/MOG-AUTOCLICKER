MOG Autoclicker

Welcome to MOG Autoclicker

A fast and reliable autoclicker with customizable features.

Download Now

Features

  • Adjustable clicks per second (CPS) from 1 to 10,000.
  • Supports right, left, and middle mouse clicks.
  • Toggle key set to 'G' for easy activation.
  • Customizable CPS/MS and randomization options.
  • User-friendly interface with status indicators.

Installation

Requirements: Python 3.11 or higher.

If the application doesn't run, open Command Prompt and execute the following command to install the necessary dependencies:

pip install keyboard mouse pyautogui tk ttkthemes pywin32 threading time messagebox ttk tkinter
