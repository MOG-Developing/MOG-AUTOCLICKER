Features

MOG-AUTOCLICKER V1

Original version with basic autoclicking functionality:

  • Left and right click support
  • Toggle key (G key by default)
  • CPS range from 1 to 10,000
  • Simple and lightweight

MOG-AUTOCLICKER V2 NEW

Improved version with advanced features:

  • Select between CPS or MS (milliseconds)
  • CPS/MS randomization
  • Left, Right, Middle click options
  • Customizable hotkeys
  • Status indicator (Stopped/Turned On)
  • Modern improved UI

How to Run

Requirements

Python 3.11 or higher (made in Python 3.11 so recommended)

Installation

If it doesn't run, open CMD (or command prompt) and run this command:

pip install keyboard mouse pyautogui tk ttkthemes pywin32 threading time messagebox ttk tkinter

This command installs all dependencies needed for the autoclicker.
