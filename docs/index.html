Welcome

Made by @misterofgames_yt!
MOG-AUTOCLICKER is a powerful and fast autoclicker that supports up to 10,000 CPS. Toggle key: G.

How to Run

Requires Python 3.11 or higher.

If it doesn't run, open CMD and run:

pip install keyboard mouse pyautogui tk ttkthemes pywin32 threading time messagebox ttk tkinter

What's New in V2

  • Select CPS or MS
  • Randomization options
  • Left, Right, Middle click
  • Custom hotkeys
  • Improved UI & Performance
